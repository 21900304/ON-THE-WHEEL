# On-The-Wheel Project



## 휠체어 이용자들을 위한 서비스 앱

- 휠체어 이용자들은 우리가 자주 이용하고 흔하게 보이는 식당들도 쉽게 갈 수 없음
- 외출하기 위해서는 좌식책상, 문턱, 엘리베이터, 낮은경사로, 장애인 화장실, 주차장이 있는지 등 수많은 환경들을 고려해야함
- 이에 대한 정보량도 부족하기 때문입니다.
